⚙ Firebase の設定が必要です

キャラクターデータをクラウドに保存するには、Firebase の設定が必要です。

  1. Firebase Console でプロジェクトを作成
  2. Firestore Database を有効化（テストモードで開始）
  3. 「プロジェクトの設定」→「マイアプリ」→「</> Web」でアプリ追加
  4. 表示された firebaseConfig の値を js/firebase-config.js に入力
  5. ファイルを保存して git push で反映
// js/firebase-config.js に以下を記入してください
const FIREBASE_CONFIG = {
    apiKey:            "ここにAPIキー",
    authDomain:        "プロジェクトID.firebaseapp.com",
    projectId:         "プロジェクトID",
    storageBucket:     "プロジェクトID.appspot.com",
    messagingSenderId: "送信者ID",
    appId:             "アプリID"
};
